# 💰 Expense Management System

A full-stack web application to track daily expenses, manage spending, and visualize financial habits.

## 🚀 Features

* ➕ Add and delete expenses
* 📂 Categorize spending
* 💾 Persistent storage using SQLite
* ⚡ Fast and responsive UI

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* Database: SQLite
